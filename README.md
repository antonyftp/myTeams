# [Epitech Network Programming] myTeams
Deuxième projet du module Network Programming d'Epitech (promo 2024)

# But du projet
Le but du projet était de réaliser un clone de l'application Microsoft Teams en langage C et en mode Shell.

# Résultat des tests

Le project est 100% fonctionnel, on peut s'y connecter à plusieurs et échanger en même temps.
L'executable serveur a été lancé sur un serveur virtuel OVH a des fins de tests, tous avec succes.
